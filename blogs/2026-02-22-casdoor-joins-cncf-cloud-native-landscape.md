The Landscape brings together widely used cloud-native projects and products; inclusion helps users and enterprises discover and evaluate solutions for building cloud-native systems.</p>

<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="about-casdoor">About Casdoor<a class="hash-link" href="https://casdoor.github.io/blog/casdoor-joins-cncf-landscape#about-casdoor" title="Direct link to About Casdoor">​</a></h2>
<div style="display: flex;"><img alt="Casdoor" src="https://cdn.casbin.org/img/casdoor-logo_1185x256.png" style="height: 80px; width: auto;" /></div>
<p>Casdoor is an open-source, UI-first Identity and Access Management (IAM) / Single Sign-On (SSO) platform with a web UI. It supports OAuth 2.0, OIDC, SAML, and CAS, as well as many social logins (Google, GitHub, WeChat, etc.), and helps you build user management and SSO in a few steps. Casdoor is developed by <a class="" href="https://casbin.org/" rel="noopener noreferrer" target="_blank">Casbin</a> in Golang and React. It uses a front-end/back-end separation architecture, supports high concurrency, and is actively maintained by the community. The project has received over 13k GitHub stars.</p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="about-cncf-and-the-cloud-native-landscape">About CNCF and the Cloud Native Landscape<a class="hash-link" href="https://casdoor.github.io/blog/casdoor-joins-cncf-landscape#about-cncf-and-the-cloud-native-landscape" title="Direct link to About CNCF and the Cloud Native Landscape">​</a></h2>
<div style="display: flex;"><img alt="CNCF - Cloud Native Computing Foundation" src="https://www.cncf.io/wp-content/uploads/2023/04/cncf-main-site-logo.svg" style="height: 80px; width: auto;" /><img alt="CNCF Cloud Native Landscape" src="https://landscape.cncf.io/images/logo_header.svg" style="height: 80px; width: auto;" /></div>
<p>The <strong>Cloud Native Computing Foundation (CNCF)</strong> is a Linux Foundation project that promotes cloud-native technologies. Its vision is to make cloud native ubiquitous. CNCF maintains the <strong>Cloud Native Landscape</strong> (<a class="" href="https://landscape.cncf.io/" rel="noopener noreferrer" target="_blank">https://landscape.cncf.io</a>), which classifies and tracks projects and products in the cloud-native ecosystem and serves as a reference map for developers and enterprises.</p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="looking-ahead">Looking ahead<a class="hash-link" href="https://casdoor.github.io/blog/casdoor-joins-cncf-landscape#looking-ahead" title="Direct link to Looking ahead">​</a></h2>
<p>Joining the CNCF Landscape reflects progress in Casdoor’s maturity and community. We will keep improving Casdoor and its integration with cloud-native workflows, and we welcome more users and contributors to join the community.</p>
