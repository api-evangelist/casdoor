<p>Welcome to Casdoor official website!👋</p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="what-is-casdoor">What is Casdoor?<a class="hash-link" href="https://casdoor.github.io/blog/welcome#what-is-casdoor" title="Direct link to What is Casdoor?">​</a></h2>
<p>Casdoor is a UI-first <a class="" href="https://en.wikipedia.org/wiki/Identity_management" rel="noopener noreferrer" target="_blank">Identity Access Management (IAM)</a> / <a class="" href="https://en.wikipedia.org/wiki/Single_sign-on" rel="noopener noreferrer" target="_blank">Single-Sign-On (SSO)</a> platform based on OAuth 2.0, OIDC, SAML and CAS.</p>
<p>Casdoor is developed by <a class="" href="https://casbin.org/" rel="noopener noreferrer" target="_blank">Casbin</a> using Golang and React.</p>
<p>Casdoor aims at providing a powerful authentication platform to handle the user management.</p>
